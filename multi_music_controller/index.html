Multi Music Controller

ESP32 (4 buttons + 1 potentiometer) → Serial → Processing (drum samples + background track + UI)

1. Overview

This project builds a simple performance controller using an ESP32 and a Processing sketch. The ESP32 sends sensor values over serial communication, and Processing uses those values to trigger sounds and control audio parameters.

B1 Kick trigger B2 Closed hi-hat trigger B3 Hi-hat trigger B4 Background track (peaches) ON/OFF toggle Pot Background volume (continuous control)

5. UI Design

  • What is being controlled: background track ON/OFF and background volume.
  • How the value changes: the volume meter and numeric value update in real time as the potentiometer rotates.
  • Event feedback: when B1–B3 are pressed, the UI briefly highlights the triggered drum sound.

6. Design Decisions

I designed this controller for solo use, like a simple personal performance tool. The interaction is easy to learn: buttons are for quick triggers and toggles, and the potentiometer is for continuous control. I focused on a light beat / drum setup (kick and hi-hats) plus one background track, so the system feels clear and playable. I placed the potentiometer outside the box for better ergonomics, while keeping the breadboard hidden inside to protect the wiring.
