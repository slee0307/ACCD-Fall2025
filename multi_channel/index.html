Multi-Channel Controller

An experimental controller using ESP32 and serial communication to manipulate multiple parameters in a Processing sketch.

Project Overview

This project explores how sensor data collected from an ESP32 can be transmitted via serial communication and mapped to multiple parameters in a Processing program.

Instead of using a potentiometer or button, alternative sensors were tested to create a richer, multi-dimensional control system.

Concept

The controller acts as a physical interface that continuously sends multiple values. These values are parsed in Processing and applied to visual parameters such as movement, color, scale, and rotation.

Build Process

The ESP32 was connected to several sensors, each producing a different range of values. These values were formatted as a single serial message and sent to Processing.

System Flow

ESP32 → Serial Communication → Processing → Visual Output

Final Result

The final system allows multiple parameters to be controlled simultaneously, creating an expressive and responsive interaction between the physical controller and the screen.

Reflection

Through this project, I learned how to structure serial data, handle multiple channels, and design meaningful mappings between physical input and digital behavior.
