Rhythmic Patterns Documentation

This page shows the three rhythmic LED patterns I created using the ESP32. Each section includes a video and a short explanation of my process.

Pattern 1 – Morse Code with My Name

Process

For this pattern, I used Morse code to show my name with LED blinking. Each dot and dash becomes a short or long blink. I wanted to try a pattern that feels personal, so I thought using my name would be interesting.

Pattern 2 – Blink with Potentiometer Speed Control

Process

In this pattern, I made a simple blink rhythm and connected a potentiometer to control the speed. When I turn the knob, the LED blinks faster or slower. This helped me understand how analog input can change the rhythm in real time and how interaction can shape the pattern.

Pattern 3 – Button Rhythm Change

Process

I used a push button to change the LED rhythm. Each press switches to a different pattern. This helped me explore how interaction can control timing and rhythm.
